#WardScript

WardScript pretende ser 2 cosas:

1. Un lenguaje de alto nivel
2. Un lenguaje de programacion util y minimalista

## Requerimientos

* make
* flex
* bison
* clang/gcc (u otro compilador de c)

## Instalacion

Instalacion en Linux:

` make `
